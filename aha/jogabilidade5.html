Qual das seguintes práticas constitui o crime de "phishing"?

Enviar mensagens falsas que se passam por empresas legítimas para obter informações confidenciais das vítimas
Invadir sistemas de computador para roubar dados pessoais através de vulnerabilidades de software
Criar vírus de computador que danificam arquivos e sistemas operacionais
Utilizar redes de computadores infectados (botnets) para realizar ataques de negação de serviço
